# chmpxpx_msgid_close
Closes a chmpx message handle

## Description

```
function chmpxpx_msgid_close($handle_res, $handle_msgid): bool {}
```

Closes a chmpx message handle.

## Parameters
* handle_res
  * Specifies a chmpx handle.
* handle_msgid
  * Specifies a chmpx message handle

## Return Values
Returns true on success or false on failure.
