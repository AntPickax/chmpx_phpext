# chmpxpx_process_running
Checks if a chmpx process is alive

## Description

```
function chmpxpx_process_running($handle_res): bool {}
```

Checks if a chmpx process is alive

## Parameters
* handle_res

## Return Values
Returns true when a chmpx process is alive or false on failure.
